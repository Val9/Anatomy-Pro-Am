= Anatomy Pro Contouring

== Setup

To run this demo you first need to install:
* NodeJS
* Node modules (npm install): connect, socket.io, redis, coffee-script, underscore
* Redis (and run the server)

Go to your folder and type:

coffee bin/app.coffee

It should be running now :-)
